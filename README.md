🔐 Projeto Cifra de César

📚 Sobre o projeto

Este projeto foi desenvolvido como atividade acadêmica com o objetivo de explorar conceitos de criptografia utilizando a clássica Cifra de César.
A proposta consistia em analisar um programa Java responsável por criptografar mensagens sem informar a chave utilizada, exigindo a aplicação de lógica de programação para descobrir a chave correta e realizar a descriptografia da mensagem.

Além da proposta original, aproveitei o projeto para implementar uma interface gráfica com Java Swing, tornando a aplicação mais intuitiva e interativa.

✨ Funcionalidades

✅ Criptografia de mensagens
✅ Descriptografia de mensagens
✅ Definição de chave de deslocamento
✅ Interface gráfica com Java Swing
✅ Manipulação automática de caracteres

🔑 Como funciona a Cifra de César

A cifra funciona deslocando cada letra do alfabeto por uma quantidade fixa de posições.

Exemplo com chave = 3

A → D
B → E
C → F

Mensagem original:

CASA

Mensagem criptografada:

FDVD
